Crag Charts

How to Use

Loader

Before creating any chart, the loader script has to be appened to the end of the HTML body.

	... ...

	<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>

</body>

... ...

Following the script, a new script tag should be added to host the chart creation and loader callback functions. The first line required is the built in function onLoadCallBack, this has 2 required variables and 1 optional, they are as follows...

crag.onLoadCallBack(callback=function, chartType=Array, src=string[Optional]);

callback The name of the function to be called once the loader has finished loading all of the required resources. This should be input without the function brackets, for example myFunction, not myFunction(). Type: function
chartType A string array of all the charts required for the current page. Example
  • ['column'] or
  • ['column', 'line']
Type: Array[string]
src A string pointing to a local source for the chart scripts. This should only be used if you want to download the files and use them localally. These files will not get the latest updates as they are released. Type: string

For example, to load the bar chart resources and call the function createCharts the following would be used

crag.onLoadCallBack(createCharts, ['column']);

Chart Creation

Once the loader has finished and initialisez the call back function, it is time to prepare and create the charts. Each chart created will need a parent element, most commonly a <div>. This element should have an id that will be passed to teh chart creation function.

<div id="bar-chart" style="width: 100%;	height: 500px;"></div>

Every chart class has a built in function of create(), this function requires a single string variable matching the parent element ID. The create function will also return the class object for the chart created to enable addditional functions to be called.

In the below example bar chart will be created and assigned to a const variable for future use.

const chart = new CragColumn(options).create('bar-chart');
Tip: For more specific chart guides, see each charts page.

In addition to passing the Id of the parent element, the class object will require an object containing specific options. At minimum, each chart will require a data series in this object, for example...

const options = {
	series: [['A', 500], ['B', 432], ['C', 300]];
}
const chart = new CragColumn(options).create('bar-chart');

See the chart specific pages on formatting and names of the ooptions available

Full Example

The below is a full working html code for a basic bar chart...

<html>
	<head>
		<title>My Chart</title>
	</head>
	<body>
		<div id="bar-chart" style="width: 100%;	height: 500px;"></div>

		<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
		<script>
			let chart;
			crag.onLoadCallBack(createCharts, ['column']);

			function createCharts() {
				const options = {
					series: [['A', 500], ['B', 432], ['C', 300]]
				}
				chart = new CragColumn(options).create('bar-chart');
			}
		</script>
	</body>
</html>
