Crag Charts

Visualization: Progress Chart

Overview

The progress chart is a single horizontal bar chart rendered in the browser using SVG and HTML, the bar prepresents the progress of a single data point against a total data point with an optional mid point target. The chart takes as minimum two values passed in the class options and requires a parent with a minimum of 250px in height.

The below if the minimum code required for a chart

<div id="progress-chart" style="width: 100%; height: 300px;"></div>

<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
<script>
	let chart;
	crag.onLoadCallBack(createCharts, ['progress']);

	function createCharts() {
		const options = {
			data: {current: 45, total: 120}
		}
		chart = new CragProgress(options).create('progress-chart');
	}
</script>

This will generate the following

Configuration Options
data.current The current progress valueType: numberDefault: 0
data.target The optional target value, this should be less than the total.Type: numberDefault: null
data.total The total progress valueType: numberDefault: null
bar.color The base color of the progress bar. See the color pallet options for available colorsType: stringDefault: blue
bar.targetColor The color the bar will change to once the target value has been reached. Type: stringDefault: null
bar.totalColor The color the bar will change to once the total value has been reached. Type: stringDefault: null
bar.striped Apply a striped overlay on the bar, this will not affect specified colors. Type: BooleanDefault: false
bar.animated Right directional animate the stiped overlay of the bar. The animation will stop when the total value has been reached. Type: BooleanDefault: false
labels.current The label of the current progress indicator.> Type: stringDefault: ‘Current’
labels.target The label of the target progress indicator. This will not show if no target value has been given.> Type: stringDefault: ‘Target’
labels.total The label of the total indicator.> Type: stringDefault: ‘Current’
