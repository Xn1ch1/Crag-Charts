Crag Charts

Visualization: Bar Chart

Overview

A bar chart is a horizontal bar chart rendered in the browser using SVG and HTML. Like all Crag charts, bar charts display tooltips when the user hovers over the data. For a vertical version of this chart, see the bar chart TODO: LINK.

The chart is limited to 20 data points, data will auomtatically be truncated if there are more points provided.

The minimum required code required for invoking a chart is the following

<div id="bar-chart" style="width: 100%;	height: 500px;"></div>

<script src="https://combinatronics.com/Xn1ch1/Crag-Charts/main/crag/crag.charts.loader.js"></script>
<script>
	let chart;
	crag.onLoadCallBack(createCharts, ['bar']);

	function createCharts() {
		const options = {
			series: [['A', 500], ['B', 432], ['C', 300]];
		}
		chart = new CragBar(options).create('bar-chart');
	}
</script>

In the second script tag, the onLoadCallBack is invoked, the function paramater passed should be below this callback that will create the chart. For more detailed information, see How to Use.

Built In Functions

Update

To update the data in the chart, the update function should be called on the chart object. Only one parameter is required, an array of arrays. See series parameter for more infomration. The data set can change in size when updated, bars will be automatically added and removed as needed but will still be limited to 20 points and truncated automatically.

Example 
chart.update([['A', 160], ['B', 527], ['C', 428]]);

Draw

The draw function will cause a complete redraw of the chart with the pre existing data. This function will be useful where the size of the container object changes size. The below example is calling the draw function on the chart object when the window resizes.

window.onresize = function() {
	chart.draw();
}

Examples

Optional paramters can be passed inside options when initializing the chart. These are used to style the chart and elements

The following example sets the bar width to 90%, sets the top of the bars to be rounded and applys an inset shadow.

let options = {
	series: [['A', 500], ['B', 436], ['C', 300], ['D', 375]],
	bar: {
		width: 90,
		rounded: true,
		inset: true
	}
}

Another example, the options passed have included vAxis and chart settings. The two paramters passed in vAxis are min and format. See TOOO LINK for more infomration on formatting axes. A min seeting of 'auto' will let the chart decide the lower bound value on the vertical axis.

In addition a chart title has also been specified and extra bar options have been passed such as color and a striped overlay.

let options = {
	series: [['A', 400], ['B', 500], ['C', 300]],
	bar: {
		width: 95,
		rounded: true,
		inset: true,
		striped: true,
		animated: true,
		color: 'amber'
	},
	vAxis: {
		min: 'auto',
		format: 'time'
	}
	chart: {
		title: 'Time Taken'
	}
}

Configuration

Options
series (Required) An array of arrays, each sub array will have the series label at index 0 and a number at index 1. A maximum of 20 points can be displayed on the chart, data will be automatically truncated. Example 
options = {
	series: [
		['A’, 5],
		['B’, 10],
		['C’, 15],
		['D’, 20],
	]
}
  Type: Array of arrays Default: null
bar.width Percentage width of each bar, where 100 means there is no space between bars. Can not exceed 100.   Type: number Default: 100
bar.color The base color of the bars. See the color pallet options for available colors . When not specified the bars will take the default array of colors .   Type: string Default: 'multi’
bar.rounded Sets the top of the bars to have rounded corners.   Type: Boolean Default: false
bar.inset Sets the bars to have an inset drop shadow.   Type: Boolean Default: false
bar.stripe Apply a striped overlay on the bars, this will not affect the specified color .   Type: Boolean Default: false
bar.animated Animate the stiped overlay of the bar.   Type: Boolean Default: false
vAxis.label The label of the vertical axis and the series values.   Type: string Default: 'Series 1’
vAxis.lines Enables horizontal major and minor gridlines on the chart for the vertical axis points   Type: string Default: true
vAxis.format Formatting of the major line values and the tool tip value label. See the label formatting guide on available formats.   Type: string Default: 'number’
vAxis.min Sets the minimum value of the vertical axis. This should not be lower than the series values. Optionally the string value 'min’ can be passed for the chart to automatically calculate the minimum value.   Type: number or string Default: 0
labels.position Sets the default position of the bar value labels. The position will automatically switch where the label cannot fit in the default position. In addition, if the labels will not fit within the series width, the labels will automatically hide.   Option
  • 'none’: labels will not be displayed
  • 'outside’: labels will appear above the bars
  • 'inside’: labels will appear inside the bars
  Type: string Default: 'none’
labels.color Sets the text color of the labels. See the color pallet options for available colors . By default, the labels will automatically switch between black and white depending on the background color .   Type: string Default: 'match’
chart.title The title to appear above the chart. See the color pallet options for available colors .   Type: string Default: null
chart.color The background color of the chart.   Type: string Default: 'white’
chart.minorLines Turns on or off the vertical axis minor lines.   Type: Boolean Default: false
